# Quickstart: EasyTicket

**Get the application running locally in under 60 seconds.**

## Prerequisites

- Docker + Docker Compose (v2) installed
- Git

## Steps

```bash
# 1. Clone the repository
git clone https://github.com/SilvaNeto29/EasyTicket.git
cd EasyTicket

# 2. Copy environment file
cp .env.example .env

# 3. Start the containers (builds on first run — ~30-60 seconds)
docker compose up -d

# 4. Install PHP dependencies
docker compose exec app composer install --no-interaction

# 5. Generate application key
docker compose exec app php artisan key:generate

# 6. Run database migrations
docker compose exec app php artisan migrate

# 7. (Optional) Install JS dependencies and build assets
docker compose exec app npm install && npm run build
```

**Open your browser**: http://localhost:8080

Register an account on the registration page and start creating projects.

## Development Workflow

```bash
# Start containers
docker compose up -d

# Stop containers
docker compose down

# Open a shell inside the container
docker compose exec app bash

# Run tests
docker compose exec app php artisan test
# or directly with Pest:
docker compose exec app ./vendor/bin/pest

# Run a specific test file
docker compose exec app ./vendor/bin/pest tests/Feature/Projects/CreateProjectTest.php

# Run Vite dev server (for CSS/JS hot reload)
docker compose exec app npm run dev

# View application logs
docker compose exec app tail -f storage/logs/laravel.log
```

## Environment Variables (`.env.example`)

```dotenv
APP_NAME=EasyTicket
APP_ENV=local
APP_KEY=                          # Generated by artisan key:generate
APP_DEBUG=true
APP_URL=http://localhost:8080

DB_CONNECTION=sqlite
DB_DATABASE=/var/www/html/database/database.sqlite

MAIL_MAILER=log                   # Emails written to log only (v1)

SESSION_DRIVER=database
SESSION_LIFETIME=120

CACHE_STORE=database
QUEUE_CONNECTION=sync
```

## Data Management

**Export your data**: Click "Export Data" on the dashboard → downloads a JSON backup file.

**Database file location**: `database/database.sqlite` (mounted in Docker volume `sqlite_data`).

**Manual backup**: `docker compose exec app cat database/database.sqlite > backup-$(date +%Y%m%d).sqlite`

**Reset everything**:
```bash
docker compose exec app php artisan migrate:fresh
```
